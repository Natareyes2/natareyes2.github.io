Html tags:

"!DOCTYPE html"
All HTML documents must start with a !DOCTYPE declaration. It is an "information" to the browser about what document type to expect.
"h1"
Heading tag that indicates the main topic on a web page.
"p"
HTML element represents a paragraph.
"ul"
This element is for grouping a collection of items that do not have a numerical ordering, and their order in the list is meaningless.
"li"
HTML element is used to represent an item in a list. Bullet point(s).
